# WebEngine Verarbeitung POC

This build adds a visual-only Verarbeitung screen to the PySide6 + QWebEngineView proof of concept.

## Scope

- HTML/CSS/JS only.
- No business logic connected yet.
- Core PDF/Excel/Mail code unchanged.
- Sidebar navigation switches between Dashboard and Verarbeitung in the WebEngine UI.

## Goal

Validate that the WebEngine architecture can reproduce the approved Verarbeitung layout much more closely than QWidget/QML.
